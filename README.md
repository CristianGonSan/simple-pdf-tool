# Simple PDF Tool

A lightweight desktop application built with **Java Swing** that lets you work with PDF files quickly and easily.

## Features

- **Merge images into a PDF** — Combine multiple images into a single PDF document.
- **Convert PDF to images** — Export each page of a PDF as an image file.
- **Combine multiple PDFs** — Merge several PDF files into one.
- **Split a PDF** — Divide a PDF into individual pages or sections.

## Download

Download the latest version for **Windows**, **macOS**, or **Linux** from the [Releases page](https://github.com/CristianGonSan/simple-pdf-tool/releases).

> Distributed and packaged using [jDeploy](https://www.jdeploy.com) — a tool for deploying Java desktop applications cross-platform with no manual JDK setup required for end users.

## Requirements (for development)

- Java 21
- Maven

## Run (development)

```bash
./mvnw exec:java
```

## Build

```bash
./mvnw package
```

## Deploy with jDeploy

This project uses [jDeploy](https://www.jdeploy.com) to publish native installers for all platforms.

**Quick deploy:**
Open the project in [jDeploy](https://www.jdeploy.com) and press **"Publish"**.

For a detailed guide, see the [jDeploy documentation](https://www.jdeploy.com/docs/manual/#_publishing_your_application).
